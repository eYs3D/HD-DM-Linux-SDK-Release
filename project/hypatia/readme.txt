Folder descriptions
========================================================================
- cfg
    Configure file for module.
- console_tester
    Source code for arm console test. Please refer to the readme file 
    in this folder to learn how to compile the console tool.
- doc
    This document describes the usage of Application Programming Interfaces of HD-DM-SDK.
- eSPDI
    This folder includ all necessnary libs and SDK header file.
- hypatia
    Cross-compiler toolchain for Hypatia project.
- out
    All files generated by the tool will be stored in this folder. 

* Please refer to the readme file in the sub folder.
